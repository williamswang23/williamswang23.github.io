Sep 2, 2021

confusion matrix

model : data–> model –> covid-19
患者：positive； 健康人： negative

10000人: positive, negative (actual)

model: prediction: positive_m; negative_m

TP: positive, positive_m: right
FP: negative, positive_m: error —— type I
FN: positive, negative_m: error —— type II
TN: negative, negative_m: right

precision (P)=TP/(TP+FP)
所有模型认为是positive的样本中，真的是positive的比例。

precision在FP成本很高的情况下很有用。

recall（R）=TP/（TP+FN）
所有真的positive，被model正确捕捉到的比例。

recall在FN成本很高的情况很有用。

accuracy= (TP+TN)/(TP+FP+TN+FN)
TP+TN: 所有被正确分类的人
TP+FP+TN+FN： 所有的人
衡量的是所有的人中，被正确分类的比例。

正确分类：
TP: positive, positive_m: right
TN: negative, negative_m: right

F1 score=(2* P *R)/(P+R)
F1 score越大越好。
