wave, correlation and rotation

raw data

the best indicator of choosing an eminent fund manager is based on risk-adjusted tools, such as SR and IR. it is un-different in most investment assessing sense but a higher precise or particular target.
the SR’s function: $SR=\frac{R_p-R_f}{\sigma_P}$;
the IR’s function: $IR=\frac{R_p-R_b}{\sigma_{R_p-R_b}}$;
whichever be chosen by our model, the outcomes are approximate.
we can use a longer-term period date to calculating portfolio return; then converting to daily return. however, we calculate the standard deviation by daily return in the same period.

filtering noise

obviously, there is much noise included in our raw data. the Kalman filtering may be an effective method to dropping noise and keeping some intensive-information trends and patterns if those patterns exist really.

wave and correlation

first of all, the rest of this essay is based on an assumption that there are some explicit patterns in daily return time series data.

wave, correlation

any anomaly return represents a signal. we can use NN tools for predicting when the next signal will happen. the probability of the next signal happening can combine a statue transforming matrix. suppose we use the data from different sectors, this matrix will reveal sector rotation patterns.
moreover, after testing the stabilization of the matrix by SVD decomposition，we can describe market sector rotation more precisely than just based on fundamental analysis.
the data snooping can portray the dynamic process of a market, although the data cannot explain the reasons behind it.
however, do we need the reasons that explain such puzzles in our mind? in my view, those reasons have only a single effect which eliminates insecurity of ourselves.
