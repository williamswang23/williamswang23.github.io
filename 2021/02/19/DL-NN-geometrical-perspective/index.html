DL&NN geometrical perspective

Feb 19, 2021

deep learning is not deep enough

single neural node

for a typical single neural node, t, it constitutes below segments:
1. weighted -$w_{t,i}$
2. bias -$b_{t}$
3. active function -discriminating linear function’s output

the linear function: $y_{t}=\sum{w_{t,i}*y_{t-1,i}}+b_{t}$
the active function : such as the ReLU function(线性整流函数 - 维基百科，自由的百科全书) has an interesting feature, which can divide a linear smooth function transform to an piecewise linear function. your can also choose alternative active

imaging a two dimension input as a vector, linear function is a affine transformation to the vector, then mapping to a 1 dimension space. then active function cuts the latter space to two segments, drops out negative one.

single layer with n nodes

we assume randomize initial weighted of each layers，which means mapping randomly original input data to a l1 dimension space then drop one half of the l1 output.

multiple layers

after transforming and mapping through multiple layers, the original data are compressed and dropped randomly until to a one hot vector or a number, which is feed-forward model’s output.

optimization process

because the ReLU function only can reserve positive segment, for minimizing loss function, the model will mapping original data with some special feature or data point to active function and pass it to next layer. that is the NN model dynamic process.

conclusion

it is a pity that i have few details of NN model, specially a complete and intuitive explanation in algebraic geometry perspective.
